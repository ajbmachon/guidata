Welcome to guidata's documentation!
===================================

.. image:: images/guidata.png
   :align: center

Based on the Qt Python binding module PyQt4, ``guidata`` is a Python library 
for easy data set edition (and display) thanks to automatically generated 
graphical user interfaces -- it also provides helpers and application 
development tools for PyQt4.

External resources:

* Downloads, bug reports and feature requests: `SourceForge`_
* Help, support and discussions around the project: `Google Group`_
  
.. _SourceForge: http://sourceforge.net/projects/guidata/
.. _Google Group: http://groups.google.fr/group/guidata_guiqwt
